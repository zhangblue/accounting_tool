# 前端CLAUDE.md

## UI开发语言

ui使用 React + Vite + TypeScript + Ant Design + Recharts进行开发

## UI展现方式

页面左侧为菜单栏（紫色背景，宽度150px），右侧为主内容区域，顶部为Header

## 项目结构

```
src/
├── pages/
│   ├── Home.tsx          # 首页组件，显示财务统计和图表
│   └── TransactionList.tsx # 账本明细组件
├── types/
│   └── index.ts          # Transaction类型定义
├── App.tsx               # 主应用组件，处理页面切换
├── App.css               # 应用样式
└── main.tsx              # 入口文件
```

## 当前菜单功能

- **首页**（默认显示）
  - 财务统计卡片（4列）
    - 全部收入（绿色）
    - 全部支出（红色）
    - 全部结余（蓝色）
    - 期初结余（黄色）
  - 图表展示
    - 左侧：饼图显示支出类型占比
    - 右侧：折线图显示每月支出趋势

- **账本** - 交易记录表格
- **统计分析** - 预留功能
- **账户分析** - 预留功能
- **我的设置** - 预留功能

## Header功能

- 显示当前页面标题
- 日期范围选择器
- "记一笔"按钮（用于快速记录交易）

## 常用命令

```bash
npm run dev      # 启动开发服务器
npm run build    # 构建生产版本
npm run lint     # 代码检查
npm run preview  # 预览生产构建
```

## 依赖库

- **antd**: UI组件库
- **recharts**: 图表库
- **react**: 前端框架
- **vite**: 构建工具

## 布局特性

- 全屏自适应显示
- Header高度固定64px
- 左侧菜单栏宽度150px
- 主内容区域自动扩展
- 支持内容滚动

## 开发注意事项

1. 交易数据存储在React state中，刷新页面会丢失（后续需接入后端API）
2. 饼图数据根据交易描述统计，若无交易则显示示例数据
3. 折线图目前使用固定的示例数据，后续需根据实际交易数据动态生成
4. 所有金额单位为元
5. 菜单项key值：1=首页, 2=账本, 3=统计分析, 4=账户分析, 5=我的设置
